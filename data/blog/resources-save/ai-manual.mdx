### Claude Code 与 Gemini

[awesome-harness-engineering](https://github.com/walkinglabs/awesome-harness-engineering)<br />
一个围绕 harness engineering 的精选资源清单，聚焦上下文管理、记忆、工作流设计、约束、安全自治、评测、可观测性和 runtime。相比泛化的 agent 资料，它更适合用来补“怎样把 Agent 跑得更稳”这一层知识。

[claude-code-ultimate-guide](https://github.com/FlorianBruniaux/claude-code-ultimate-guide)<br />
一份面向 Claude Code 的系统化学习仓库，覆盖从入门到进阶的工作流设计、上下文管理、多智能体协作、安全风险、模板示例和知识测验。适合拿来建立完整认知，而不是只抄配置。

[claude-code-book](https://github.com/lintsinghua/claude-code-book)<br />
一本围绕 Claude Code 的 Agent Harness 架构分析书，不讲提示词技巧，重点拆解对话循环、工具系统、权限管线、上下文压缩、子智能体、MCP 集成和技能插件等核心机制。

[Harness Books](https://harness-books.agentway.dev/index.html)<br />
一个围绕 Harness Engineering 组织的在线书籍入口，目前收了两本主线内容：一本聚焦 Claude Code 的运行时骨架，另一本横向比较 Claude Code 和 Codex 的 harness 设计哲学。适合在读完 `claude-code-book` 之后，继续补控制面、状态治理、恢复路径、验证分工和本地规则这些更偏“系统设计判断”的内容。
