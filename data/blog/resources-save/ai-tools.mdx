## 生活

### [ebook-to-mindmap](https://github.com/SSShooter/ebook-to-mindmap)

这是一个基于 AI 技术的智能电子书解析工具，支持将 EPUB 和 PDF 格式的电子书转换为结构化的思维导图和文字总结。工具采用 BYOK（自带 API 密钥）模式保障隐私，提供多种处理模式，并具备缓存管理、现代化界面等特性

### [由 AI 生成的书籍深度总结](https://bookds.com/)

BookDS · Book Deep Summary 是一个利用先进 AI Agent 对书籍进行深度总结的服务，旨在帮助用户快速掌握书籍核心内容。它强调 AI 总结的辅助性质，并倡导用户支持正版书籍。

## 演示文稿

### [banana-slides](https://github.com/Anionex/banana-slides)

这是一个基于 nano banana pro 模型的原生 AI PPT 生成应用，旨在通过多种灵活的创作路径、强大的素材解析能力以及自然语言交互，帮助用户快速生成美观且专业的演示文稿。项目支持从想法、大纲或页面描述开始生成完整 PPT，并允许通过口头指令进行修改，同时提供可编辑的 PPTX 导出功能。它适用于各类用户，包括学生、职场人士和教育工作者，致力于降低 PPT 制作门槛并提升设计自由度。

## 垂类 Agent

### [Open-AutoGLM](https://github.com/zai-org/Open-AutoGLM)

这是一个基于 AutoGLM 构建的手机端智能助理框架。它能够通过多模态模型理解手机屏幕内容，并借助 ADB/HDC 自动化操作帮助用户完成任务。文档详细说明了项目的安装部署、环境配置、模型服务选择（支持第三方 API 或本地部署）、远程调试方法以及二次开发指南，同时强调了项目的学习研究用途，并提供了常见问题解决方案。

### [midscene](https://github.com/web-infra-dev/midscene)

Midscene.js 是一个基于视觉模型驱动的跨平台 UI 自动化框架，支持通过自然语言描述自动化任务，覆盖 Web、Android、iOS 等多种界面，并提供开发者友好的 SDK、调试工具和社区生态。

## CodeAgent 伴侣

### [dotagents](https://github.com/iannuttall/dotagents)

这里可以集中管理您的所有钩子函数、命令行工具、智能技能以及 AGENT/CLAUDE.md 配置文件。

### [osaurus](https://github.com/dinoki-ai/osaurus)

Osaurus 是一款专为 macOS（Apple Silicon）设计的 AI 边缘运行时，集本地与云端模型于一体。它通过 MCP 协议暴露共享工具，为 AI 应用和工作流提供了一个原生、常驻的基础平台。核心功能包括运行本地模型、连接多种远程 API、支持 MCP 服务器与客户端、可扩展的插件系统、自定义 AI 助手（Personas）、任务自动化调度、多窗口聊天、开发者工具以及强大的本地语音输入与转录功能。

### [happy](https://github.com/slopus/happy)

Happy Coder，这是一个为 Claude Code 和 Codex AI 编程工具提供的移动端与网页客户端，支持端到端加密和跨设备即时切换。

### [hapi](https://github.com/tiann/hapi)

本文介绍了 HAPI（哈皮）项目，这是一个本地优先的 AI 编程辅助工具，支持在本地运行 Claude Code、Codex 和 Gemini 等 AI 会话，并可通过 Web、PWA 或 Telegram Mini App 进行远程控制。项目强调无缝切换、远程审批、实时进度跟踪和多模型支持等特性，同时提供了从安装到使用的详细指南。

### [ccmate](https://randynamic.org/ccmate)

CC Mate 是一款为 Claude Code 设计的图形化配置与管理工具，提供直观的界面来轻松设置、切换配置、监控使用情况并管理 MCP 服务器。

### [cc-cli](https://github.com/cjh-store/cc-cli)

CC CLI 是一个用于管理 Claude Code 和 Codex 配置的命令行工具，支持一键切换 API 站点与 Token、智能合并配置、WebDAV 云端备份等功能，简化配置流程。

### [CodeBar-用量统计](https://github.com/steipete/CodexBar)

CodexBar 是一款专为 macOS 14+ 设计的轻量级菜单栏应用，可实时监控并显示多种 AI 服务（如 Codex、Claude、Gemini 等）的用量限额和重置时间。它支持多服务切换、隐私优先设计，并提供 CLI 工具和桌面小组件，方便用户管理使用情况。

## CodeAgent 代码编辑器

### [conductor](https://conductor.build/)

该工具允许用户在隔离的工作空间中并行创建 Codex 和 Claude Code 代理，实时查看其工作状态，并审核与合并代码更改。文章还说明了其操作流程、常见问题及费用机制。

### [EnsoAI](https://github.com/J3n5en/EnsoAI)

EnsoAI 是一款专为开发者设计的 AI 智能工作空间管理工具，通过集成 Git Worktree 与多 AI 代理（如 Claude、Gemini、Codex 等），支持在同一个项目中并行处理多个分支任务，无需频繁切换上下文。它提供轻量级的代码编辑、可视化 Git 管理、AI 代码审查及冲突合并等功能，旨在提升多任务开发效率。
