## 生活

[OpenManus](https://github.com/FoundationAgents/OpenManus)  
OpenManus 是一个开源通用 AI 代理框架，无需邀请码即可使用，支持快速构建和定制 AI 代理。它提供简易安装、多代理集成、强化学习扩展及社区协作功能，旨在帮助用户高效实现各类 AI 应用创意。

[clawdbot](https://github.com/clawdbot/clawdbot)  
Clawdbot 是一款可在本地设备上运行的个人 AI 助手，支持通过 WhatsApp、Telegram、Slack、Discord、Google Chat、Signal、iMessage、Microsoft Teams、WebChat 等多种常用通讯渠道进行交互，并具备语音唤醒、实时画布、多智能体路由等功能。其核心是一个本地优先的网关控制平面，用户可通过 CLI、Web 界面或配套应用进行管理和交互。

[ebook-to-mindmap](https://github.com/SSShooter/ebook-to-mindmap)

这是一个基于 AI 技术的智能电子书解析工具，支持将 EPUB 和 PDF 格式的电子书转换为结构化的思维导图和文字总结。工具采用 BYOK（自带 API 密钥）模式保障隐私，提供多种处理模式，并具备缓存管理、现代化界面等特性

[由 AI 生成的书籍深度总结](https://bookds.com/)

BookDS · Book Deep Summary 是一个利用先进 AI Agent 对书籍进行深度总结的服务，旨在帮助用户快速掌握书籍核心内容。它强调 AI 总结的辅助性质，并倡导用户支持正版书籍。

[short-video-factory - 短视频工厂](https://github.com/YILS-LIN/short-video-factory)  
短视频工厂是一款开源桌面应用，通过 AI 技术实现一键生成产品营销与泛内容短视频，支持自动化剪辑、多语言合成及跨平台使用，旨在降低视频制作门槛。

## 演示文稿

[banana-slides](https://github.com/Anionex/banana-slides)

这是一个基于 nano banana pro 模型的原生 AI PPT 生成应用，旨在通过多种灵活的创作路径、强大的素材解析能力以及自然语言交互，帮助用户快速生成美观且专业的演示文稿。项目支持从想法、大纲或页面描述开始生成完整 PPT，并允许通过口头指令进行修改，同时提供可编辑的 PPTX 导出功能。它适用于各类用户，包括学生、职场人士和教育工作者，致力于降低 PPT 制作门槛并提升设计自由度。

[AionUi](https://github.com/iOfficeAI/AionUi)

免费、本地化、开源的协作工具，支持 Gemini CLI、Claude Code、Codex、Opencode、Qwen Code、Goose Cli、Auggie 等多种平台

## Agent 平台

[Auto-Claude](https://github.com/AndyMik90/Auto-Claude)

## 垂类 Agent

[UI-TARS-desktop](https://github.com/bytedance/UI-TARS-desktop)

[Open-AutoGLM](https://github.com/zai-org/Open-AutoGLM)

这是一个基于 AutoGLM 构建的手机端智能助理框架。它能够通过多模态模型理解手机屏幕内容，并借助 ADB/HDC 自动化操作帮助用户完成任务。文档详细说明了项目的安装部署、环境配置、模型服务选择（支持第三方 API 或本地部署）、远程调试方法以及二次开发指南，同时强调了项目的学习研究用途，并提供了常见问题解决方案。

[midscene](https://github.com/web-infra-dev/midscene)

Midscene.js 是一个基于视觉模型驱动的跨平台 UI 自动化框架，支持通过自然语言描述自动化任务，覆盖 Web、Android、iOS 等多种界面，并提供开发者友好的 SDK、调试工具和社区生态。

## CodeAgent 编排并行

[gastown](https://github.com/steveyegge/gastown)  
Gas Town 是一个多智能体编排系统，专为 Claude Code 设计，通过 Git 支持的钩子实现持久化工作跟踪。它允许用户协调多个智能体处理不同任务，解决智能体重启后上下文丢失、手动协调困难以及工作状态易失等问题，支持扩展到 20-30 个智能体协同工作。

[CodexMonitor](https://github.com/Dimillian/CodexMonitor)  
CodexMonitor 是一款基于 Tauri 框架的 macOS 应用程序，用于在本地工作区中编排和管理多个 Codex 智能体。它通过侧边栏管理项目，提供主屏幕快速操作，并支持基于 Codex app-server 协议的对话视图。

[worktrunk](https://github.com/max-sixty/worktrunk)  
Worktrunk 是一款专为并行 AI 代理工作流设计的 Git 工作树管理命令行工具。

## CodeAgent 增强

[agent-browser](https://github.com/vercel-labs/agent-browser)  
本文介绍了 `agent-browser`，这是一个专为 AI 智能体设计的无头浏览器自动化命令行工具。它基于 Rust 开发，支持 Node.js 回退方案，提供丰富的命令集，用于网页导航、元素交互、截图、网络控制等自动化操作。工具强调通过“快照（snapshot）”与“引用（refs）”机制优化 AI 工作流，并支持多会话管理、自定义浏览器、流式传输等高级功能。

[skills-npm](https://github.com/antfu/skills-npm)  
skills-npm 是一个命令行工具，用于发现 npm 包中内置的智能体技能，并通过创建符号链接供编程智能体使用。该项目旨在解决现有技能分发方式中的摩擦，如仅支持 Git 源、版本不匹配和手动管理问题，提出将技能直接打包在 npm 包中的新约定。

[skillman](https://github.com/pi0/skillman)  
本文介绍了 skillman 工具，它是一个用于管理项目 Agent 技能的命令行工具，通过读取和操作 skills.json 文件来安装、添加来自不同来源的技能。

[claude-mem](https://github.com/thedotmack/claude-mem)  
Claude-Mem 是一款为 Claude Code 设计的持久化记忆压缩系统，能够自动捕获工具使用记录、生成语义摘要，并在不同会话间保持上下文连续性，从而让 Claude 在项目开发中具备跨会话的记忆能力。

[cc-mirror](https://github.com/numman-ali/cc-mirror)  
CC-MIRROR 是一个用于创建和管理独立 Claude Code 实例的工具，支持多种模型提供商，提供预设配置、主题和技能包，并包含旧版团队协作功能（仅限 1.6.3 版本）。用户可通过命令行快速创建隔离的 Claude Code 变体，每个变体拥有独立的配置和会话。

[rulesync](https://github.com/dyoshikawa/rulesync)  
Rulesync 是一个 Node.js 命令行工具，旨在通过统一的 AI 规则文件，自动为多种主流 AI 开发工具生成配置文件。它支持选择性生成、全面的导入/导出功能，并能管理规则、命令、MCP 配置、忽略文件、子代理和技能，从而实现跨工具开发规则的一致性

[dotagents](https://github.com/iannuttall/dotagents)  
这里可以集中管理您的所有钩子函数、命令行工具、智能技能以及 AGENT/CLAUDE.md 配置文件。

[osaurus](https://github.com/dinoki-ai/osaurus)  
Osaurus 是一款专为 macOS（Apple Silicon）设计的 AI 边缘运行时，集本地与云端模型于一体。它通过 MCP 协议暴露共享工具，为 AI 应用和工作流提供了一个原生、常驻的基础平台。核心功能包括运行本地模型、连接多种远程 API、支持 MCP 服务器与客户端、可扩展的插件系统、自定义 AI 助手（Personas）、任务自动化调度、多窗口聊天、开发者工具以及强大的本地语音输入与转录功能。

[happy](https://github.com/slopus/happy)  
Happy Coder，这是一个为 Claude Code 和 Codex AI 编程工具提供的移动端与网页客户端，支持端到端加密和跨设备即时切换。

[hapi](https://github.com/tiann/hapi)  
本文介绍了 HAPI（哈皮）项目，这是一个本地优先的 AI 编程辅助工具，支持在本地运行 Claude Code、Codex 和 Gemini 等 AI 会话，并可通过 Web、PWA 或 Telegram Mini App 进行远程控制。项目强调无缝切换、远程审批、实时进度跟踪和多模型支持等特性，同时提供了从安装到使用的详细指南。

[ccmate](https://randynamic.org/ccmate)  
CC Mate 是一款为 Claude Code 设计的图形化配置与管理工具，提供直观的界面来轻松设置、切换配置、监控使用情况并管理 MCP 服务器。

[cc-cli](https://github.com/cjh-store/cc-cli)  
CC CLI 是一个用于管理 Claude Code 和 Codex 配置的命令行工具，支持一键切换 API 站点与 Token、智能合并配置、WebDAV 云端备份等功能，简化配置流程。

[CodeBar-用量统计](https://github.com/steipete/CodexBar)  
CodexBar 是一款专为 macOS 14+ 设计的轻量级菜单栏应用，可实时监控并显示多种 AI 服务（如 Codex、Claude、Gemini 等）的用量限额和重置时间。它支持多服务切换、隐私优先设计，并提供 CLI 工具和桌面小组件，方便用户管理使用情况。
