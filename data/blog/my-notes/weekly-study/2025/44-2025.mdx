## 技术

### [元素：setHTML() 方法 - Web API 接口 | MDN](https://developer.mozilla.org/en-US/docs/Web/API/Element/setHTML)

Element 接口的 setHTML() 方法提供了一种 XSS 安全的方式，用于解析和清理 HTML 字符串并将其作为元素的子树插入 DOM 中。

- 🛡️ 该方法会移除所有 XSS 不安全元素和属性，即使这些元素被传入的清理器允许也会被移除
- ⚠️ 始终会被移除的元素包括：`<script>`、`<frame>`、`<iframe>`、`<object>`、`<use>`、事件处理属性和数据属性
- 🔧 支持可选参数 options，可传入 Sanitizer 对象或 SanitizerConfig 配置对象
- 📝 建议作为 Element.innerHTML 的安全替代方案，特别是在处理用户提供的 HTML 时
- 🧪 目前属于实验性技术，浏览器兼容性有限，生产环境使用前需仔细检查兼容性表
- 🚫 即使清理器配置允许不安全选项，该方法仍会移除 XSS 不安全实体
- 💡 提供了默认清理器和自定义清理器的使用示例，展示不同配置下的清理效果

### [XCodeReviewer](https://github.com/lintsinghua/XCodeReviewer)

XCodeReviewer 是一个由大型语言模型（LLM）驱动的智能代码审计平台，旨在通过 AI 技术为开发者提供全面、深度的代码质量分析与审查服务。它支持多平台 LLM 集成、可视化配置管理，并提供从安全性、性能到可维护性等多维度的代码评估，帮助用户快速识别和修复代码问题。

- 🚀 **AI 驱动的深度分析**：利用 LLM 理解代码意图，发现深层逻辑问题，超越传统静态分析工具。
- 🌐 **多平台 LLM 支持**：集成 10+ 主流 LLM 平台（如 Gemini、OpenAI、Claude、通义千问等），支持用户自由配置和切换。
- ⚙️ **可视化运行时配置**：无需重新构建镜像，直接在浏览器中配置 LLM 参数和 API Keys，支持 API 中转站，配置保存在本地。
- 📊 **多维度代码评估**：从安全性、性能、可维护性到代码风格，提供 360 度无死角的质量评估。
- 💡 **可解释性修复建议**：采用 What-Why-How 模式，清晰指出问题、解释原因，并提供具体修复代码示例。
- 🔧 **核心功能丰富**：包括项目管理、即时分析、智能审计、报告导出等，支持多语言代码仓库集成。
- 🐳 **便捷的部署方式**：推荐使用 Docker 部署，支持本地开发模式，并提供详细的环境变量配置说明。
- 📈 **专业审计报告**：支持导出 JSON 和 PDF 格式报告，包含完整问题详情和修复建议。
- 🛡️ **隐私与安全警告**：用户需注意代码隐私，避免上传敏感信息，建议使用本地模型或私有部署服务。
- 🔮 **未来开发计划**：包括 CI/CD 集成、多智能体协作、审计标准自定义等功能的加急开发。

### [AI-Trader](https://github.com/HKUDS/AI-Trader)

AI-Trader 是一个多 AI 模型自主交易竞赛平台，让五个不同的 AI 模型在纳斯达克 100 股票市场上进行完全自主的交易竞争，目标是测试哪个 AI 能在零人工干预下获得最高收益。

📈 项目概览总结

- 🏆 **当前冠军排行榜**：DeepSeek 以 +12.94% 收益领先，MiniMax-M2(+8.56%) 和 GPT-5(+6.87%) 分列二三位
- 🤖 **完全自主决策**：AI 代理进行 100% 独立分析、决策和执行，无人为干预
- 🛠️ **纯工具驱动架构**：基于 MCP 工具链，AI 通过标准化工具调用完成所有交易操作
- ⏰ **历史回放功能**：支持任意时间段的市场数据回放，自动过滤未来信息
- 💰 **统一竞赛条件**：所有 AI 模型从 1 万美元起步，交易纳斯达克 100 成分股

🔧 核心功能特点

- 🏆 **多模型竞技场**：部署 GPT、Claude、Qwen 等多个 AI 模型进行竞争性交易
- 📊 **实时性能分析**：完整的交易记录、持仓监控和盈亏分析
- 🔍 **智能市场情报**：集成 Jina 搜索获取实时市场新闻和财经报告
- ⚡ **MCP 工具链集成**：基于模型上下文协议的模块化工具生态系统
- 🔄 **可扩展策略框架**：支持第三方策略和自定义 AI 代理集成

🚀 技术架构

- 🤖 **核心系统**：main.py 为主程序入口，agent/base_agent 为 AI 代理核心
- 🛠️ **MCP 工具链**：包含交易执行、价格查询、信息搜索、数学计算等工具
- 📊 **数据系统**：完整的 OHLCV 数据、交易记录和性能指标
- 🎨 **前端界面**：Web 仪表板展示实时性能跟踪

📋 快速开始

- ⚡ **一键安装**：克隆项目、安装依赖、配置环境变量
- 🔑 **环境配置**：需要 OpenAI、Alpha Vantage、Jina AI 的 API 密钥
- 🎮 **运行步骤**：数据准备→启动 MCP 服务→开始 AI 竞技场

🌟 即将更新

- ⏰ **小时级交易支持**：升级到时级精度交易

🔄 项目特色
